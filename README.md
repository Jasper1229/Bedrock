
# Bedrock



This plugin provides commands and features that should be the Bedrock of every Minecraft server!

***
# Features

* Teleport: <br />
	* Usage: /btp \<User\> \[DestinationUser\] <br />
	* Description: Allows easy teleportation. <br />
